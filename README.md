# SafeExtensions

Monitor your browser extensions for privacy risks.

## Installation
1. Download from Chrome Web Store (link coming soon)
2. Click extension icon to see privacy scores
3. Disable risky extensions with one click

## Privacy
100% local. Zero data collection. Open source.

## How It Works
- Scans all installed extensions
- Analyzes permissions against known risky patterns
- Provides 0-10 privacy score
- One-click disable/uninstall controls

## Features
- ✅ Real-time extension scanning
- ✅ Privacy risk scoring
- ✅ Permission analysis
- ✅ Disable/uninstall controls
- ✅ Detailed permission view
- ✅ Local storage only

## Testing
Load unpacked extension in Chrome:
1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this folder
