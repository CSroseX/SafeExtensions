# User Guide

## Installation
1. Visit Chrome Web Store
2. Click "Add to Chrome"
3. Grant permissions when prompted

## How to Use
1. Click extension icon in toolbar
2. View privacy scores for all extensions
3. Click "View Details" for more info
4. Click "Disable" for risky extensions

## Understanding Scores
- 9-10 (A+/A): Safe to use
- 7-8 (A): Low risk
- 5-6 (B): Moderate caution
- 3-4 (C): High risk
- 0-2 (D/F): Dangerous - disable immediately

## FAQ
**Q: Why is my favorite extension flagged?**
A: We flag risky permissions, not malicious intent. Some legitimate extensions need broad permissions to function.

**Q: Can I whitelist extensions?**
A: Coming in v2.0!

**Q: Does this slow down my browser?**
A: No. Scans run in the background with minimal impact.
