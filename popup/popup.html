Important Disclaimer

Please read carefully

This is NOT a virus or malware scanner

SafeExtensions is a privacy analysis tool that helps you understand the permissions and potential privacy risks of your installed Chrome extensions.

What this tool does:

  • Analyzes extension permissions and access patterns
  • Identifies potential privacy risks based on permission combinations
  • Provides transparency about what your extensions can access
  • Helps you make informed decisions about your extensions

What this tool does NOT do:

  • Scan for viruses, malware, or malicious code
  • Guarantee the safety or security of any extension
  • Replace antivirus or security software
  • Monitor real-time extension behavior

Note: A "risky" rating indicates that an extension has broad permissions that could be misused. This doesn't mean the extension is malicious. Always install extensions from trusted developers and the official Chrome Web Store.
